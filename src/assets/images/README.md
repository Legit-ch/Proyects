# Image Asset Placeholders

This directory contains custom image assets for the NativeScript application.

## Custom Icons Added:
- User avatar placeholder
- Product images placeholders
- Custom app launcher icons (Android adaptive icons)

## Directory Structure:
- phone.png - Smartphone product image
- headphones.png - Headphones product image
- shoes.png - Running shoes product image
- coffee.png - Coffee maker product image
- laptop.png - Gaming laptop product image
- user-avatar.png - User profile avatar

These images would typically be actual PNG/JPG files in a real application.
The Android adaptive icons are implemented as vector drawables in the App_Resources/Android directory.
